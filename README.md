本项目是一个简单的番茄你闹钟，分为番茄和任务两部分：在番茄部分可以定义一个番茄闹钟，并查阅番茄历史；在任务部分可以自己新建任务并查看任务历史，此外具有登录和注册功能，可以同步番茄信息，在 PC 端和移动端均具有良好的适配性。

可以 [点击这里](https://hais-teatime.com/potado/) 在线预览。

本项目是一个基于 React 的简单单页应用，并且使用 TypeScript 进行书写，使用 axios 发送请求、React-Router 来进行页面跳转、Redux 管理任务信息，中途也遇到了不少麻烦，我将遇到的部分问题与解决思路放在了 [这篇博客](https://hais-teatime.com/post/2020-01-30-potado/) 里面，未来我还会继续总结关于 TypeScript 和 React 相关的问题，届时将会更新更多的博客内容。
